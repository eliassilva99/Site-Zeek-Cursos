O QUE VOCÊ VAI APRENDER?



Lógica de Programação


O curso abrange conceitos como algoritmos, estruturas de controle de fluxo, tipos de dados, variáveis e lógica condicional. Os participantes aprendem a criar soluções lógicas para problemas específicos e a traduzir essas soluções para linguagens de programação. Ao concluir o curso, os alunos estarão aptos a compreender a estrutura lógica por trás da programação, facilitando a transição para aprendizados mais avançados em linguagens específicas e preparando-se para atuar em áreas como desenvolvimento de software e programação de sistemas.

Desenvolvimento WEB - HTML


O curso abrange tópicos como tags HTML, semântica, listas, formulários e incorporação de mídia. Os participantes também exploram boas práticas de acessibilidade e otimização para motores de busca. Ao concluir o curso, os alunos estarão aptos a construir páginas web estruturadas e acessíveis, proporcionando uma base sólida para o desenvolvimento web e preparando-se para carreiras na área de criação de conteúdo online.

Desenvolvimento WEB - CSS


O curso abrange conceitos como seletores, propriedades de estilo, layout responsivo, e design visual. Os participantes também exploram técnicas avançadas de estilização, como animações e transições. Ao concluir o curso, os alunos estarão aptos a criar layouts atraentes, consistentes e responsivos, aprimorando suas habilidades no desenvolvimento web e preparando-se para atuar na criação estética e funcional de páginas online.

Desenvolvimento WEB - JavaScript


O curso abrange conceitos como manipulação dinâmica de conteúdo HTML, interatividade do usuário, controle de eventos, e integração com APIs. Ao concluir o curso, os alunos estarão aptos a criar funcionalidades interativas e dinâmicas em páginas web, contribuindo para suas habilidades em desenvolvimento front-end e preparando-se para carreiras na área de programação web.

Wordpress


O curso abrange tópicos como criação de páginas e postagens, personalização de design com plugins, e otimização para mecanismos de busca. Os participantes também podem explorar aspectos práticos, como a gestão de usuários e a segurança do site. Ao concluir o curso, os alunos estarão capacitados a desenvolver websites funcionais e atrativos usando o WordPress, preparando-se para atuar em design web, marketing digital ou administração de conteúdo online.

Adobe Photoshop - Essencial


Com o Photoshop, você aprenderá a manipular imagens de maneira profissional. Desde a edição básica, ajuste de cores e recorte, até técnicas avançadas como retoque de fotos e criação de designs impressionantes, o Photoshop é uma ferramenta essencial para aprimorar e personalizar imagens para diversos fins.

Adobe Photoshop - Avançado


O curso de Photoshop Avançado é projetado para usuários que buscam aprofundar seus conhecimentos em edição de imagens e design digital. Os participantes aprenderão técnicas avançadas de retoque de fotos, manipulação de camadas, uso eficiente de máscaras, criação de efeitos especiais e composições complexas. O curso aborda também a integração de ferramentas 3D, automação de tarefas com scripts, e estratégias avançadas para trabalhar com diferentes tipos de projetos. Ao final do curso, os alunos estarão aptos a realizar edições sofisticadas e criar composições visuais impactantes no Adobe Photoshop.
